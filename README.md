## Project Overview

Customer churn refers to customers leaving a company's services. This project predicts whether a telecom customer is likely to churn using Machine Learning classification algorithms.

## Problem Statement

The Telecom Customer Churn dataset contains information about customers from a telecommunications company.

The objective is to identify customers who are likely to leave the company so that proactive retention strategies can be applied.

---

## Dataset Information

Total Customers: 7043

Dataset Features Include:

- Customer demographics
- Service subscriptions
- Contract details
- Monthly charges
- Total charges
- Churn status
